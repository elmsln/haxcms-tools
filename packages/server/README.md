# Usage
<!-- usage -->
```sh-session
$ npm install -g @hax-tools/server
$ hax-server COMMAND
running command...
$ hax-server (-v|--version|version)
@hax-tools/server/0.0.0 darwin-x64 node-v10.12.0
$ hax-server --help [COMMAND]
USAGE
  $ hax-server COMMAND
...
```
