# Usage
<!-- usage -->
```sh-session
$ npm install -g @haxcms/swarm-boilerplate
$ swarm-boilerplate COMMAND
running command...
$ swarm-boilerplate (-v|--version|version)
@haxcms/swarm-boilerplate/0.0.3 darwin-x64 node-v10.12.0
$ swarm-boilerplate --help [COMMAND]
USAGE
  $ swarm-boilerplate COMMAND
...
```
